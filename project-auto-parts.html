Auto Parts Management System

This is a C# and MySQL desktop application that allows customers to browse and order car parts. It includes a full admin system for inventory, orders, users, and reports.

  • Developed using Windows Forms in C#
  • Connected to MySQL using phpMyAdmin
  • Includes user login, order processing, and report export features

Technologies: C#, MySQL, .NET Framework
